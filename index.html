NAVPRAYAS - Simple

Simple is a one page Simple Bootstarp theme for Bootstarp created by Vikash Pateshwari. This theme features several content sections, a responsive grid with hover effects, full page Gallery item added, and a working PHP contact form.

Getting Started

To use this theme, choose one of the following options to get started:

  • Download the latest release on Start Bootstrap
  • Fork this repository on GitHub

Bugs and Issues

Have a bug or an issue with this theme? Open a new issue here on GitHub or leave a comment on the template overview page at Start Bootstrap.
